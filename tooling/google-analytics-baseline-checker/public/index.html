Google Analytics Baseline Checker

A tool to help Google Analytics users determine the best Baseline target for their site, based on their users.

How it works

Step 1:
Export your browser usage data from Google Analytics (learn how).
Step 2:
Import that data using the tool below to generate a report.
Step 3:
Use this report to help you choose the best Baseline target for you.

Import your Google Analytics data

Or view an example report based on data from web.dev.

Note: All data imported into this tool is processed entirely on device. No data is processed or stored remotely.

Google Analytics Export Instructions

To learn how to export your browser usage data from Google Analytics, watch this screencast or follow the instructions below:

  1. Log into Google Analytics.
  2. Choose the account from the dropdown menu at the top that corresponds to the website you want to export browser data from.
  3. Select the Explore link from the left-hand sidebar.
  4. If you've already created a Baseline export exploration, select it from the list of explorations and skip to step #12.
  5. Select the Blank option to create a new free-form exploration.
  6. Under Exploration name enter a descriptive name like Baseline export.
  7. Select a date range (the default Last 28 days range is recommended).
  8. Click the add dimension button next to Dimensions and select the following five dimensions (found under Platform / device), then click Confirm to add them to your exploration: Browser, Browser version, Device category, Operating system, OS version.
  9. Click the add metrics button next to Metrics and select the Active users (found under Users), then click Confirm to add it to your exploration:
  10. Under Rows in the next column, click or drag to add all five of the dimensions chosen in step #8.
  11. Under Values further down in the same column, click or drag to add the Active users metric chosen in step #9. This will generate the initial report table.
  12. In the top right corner of the report click the export data button and choose TSV to save the file.
  13. Once the file is saved, import the file into this tool using the widget above.
